ðŸ§  Algorithm Logic: SRTF (Preemptive SJN)

Key Principle:

SRTF is a preemptive scheduling algorithm. It always selects the process in the ready queue with the minimum remaining execution time to run next.

Preemption Rule:

Preemption occurs at any time $t$ if a newly arriving process has a remaining burst time that is shorter than the remaining burst time of the currently running process.
